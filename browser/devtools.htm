開發人員工具

瀏覽器開發者工具（DevTools）就像是網頁開發者的「萬用工具箱」。它像是一個 「網頁的 X 光機」，可以完全透視網頁的內容 ( 包含看得到跟看不到的部分 )，並且內建在 Chrome、Edge、Firefox、Safari 等主流瀏覽器中，讓你不需要開啟網頁編輯工具，不需要修改原始碼，就能直接在瀏覽器上即時測試、除錯與最佳化網頁的呈現。

你可以點在網頁上透過滑鼠右鍵選按右鍵 → 檢查（Inspect），或是直接按鍵盤：

  • Windows: 鍵盤 F12 或 Ctrl + Shift + I
  • MacOS: 鍵盤 Cmd + Option + I (備註)

備註： Safari 的開發工具預設是關閉的，第一次使用需要先到「偏好設定」裡選取「進階」，再勾選「在功能表列中顯示開發功能表」方塊，就可以再跟瀏覽器選單中找到「開發功能」。

常用功能面板

  • Device Toolbar（手機 / 平板模式）
    模擬檢視網頁在不同尺寸，不同裝置上的的表現。通常用在響應式網站設計。
  • Elements（元素）
    可以查看與即時編輯 HTML 結構，觀察 CSS Box Model ，修改與測試 CSS 樣式，並且馬上看的到畫面變化 ( 但無法存檔 )。
  • Console（主控台）
    顯示 JavaScript 錯誤與警告，以及透過 Console 直接輸入 JavaScript 指令進行測試。
  • Sources（原始碼）
    查看 JavaScript 原始碼，或設中斷點（breakpoint）來 逐步除錯。
  • Network（網路）
    查看所有網路請求（API、圖片、CSS、JS），分析請求時間、回傳資料、是否成功。
  • Application / Storage（應用程式）
    查看 Cookie、LocalStorage、SessionStorage，管理快取、PWA （Progressive Web App）資源。
  • Performance / Lighthouse（效能）
    分析網頁效能瓶頸，檢查載入速度、a11y 無障礙存取性 ( Accessibility )、SEO 最佳化建議等。
