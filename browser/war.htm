瀏覽器大戰

世界上第一個瀏覽器，是全球資訊網的發明人 Tim Berners-Lee 於 1990 年在歐洲核子研究組織 (CERN) 所發明的。當時的瀏覽器名稱就叫做 WorldWideWeb ( 後來為了避免與 WWW 全球資訊網混淆而改名為 Nexus )。

1991 年至 1992 年間，多種瀏覽器相繼問世，例如 Line Mode Browser（第一個可跨平台瀏覽的工具）、Erwise、ViolaWWW 和 MidasWWW ⋯等。雖然看似不少選擇，但這些初期的瀏覽器都只是簡單的 HTML 文件檢視器，圖片等其他多媒體內容還是得依賴外部應用程式來檢視。

1993 年由美國 國家超級電腦應用中心（NCSA）開發的 Mosaic 成為關鍵轉折點。它是 第一個能在文字中直接顯示圖像的圖形介面瀏覽器，極大地推動了網際網路的普及，被譽為網路時代的「殺手級應用」。

後來 Mosaic 的核心開發者馬克·安德森（Marc Andreessen）創立了 Netscape 網景公司，於 1994 年發布 Netscape Navigator 瀏覽器。到1995年中期，Netscape 佔據了全球約 80% 的市場份額。

第一次瀏覽器大戰

微軟於 1995 年推出 Internet Explorer (IE) 1.0 瀏覽器。為了與當時的 Netscape Navigator 抗衡，微軟利用其在作業系統市場的壟斷地位，將 IE 免費捆綁在 Windows 系統中，讓使用者無需額外下載即可使用瀏覽器，同時也點燃了第一次瀏覽器大戰的戰火。

由於微軟強大的財力和市場優勢，Netscape 的市佔率快速下滑，最終在 1999 年被美國線上（AOL）收購。2002 年，IE 的全球市場份額達到 95% 的高峰，幾乎成為進入網際網路的唯一入口。

微軟的捆綁行為引發了著名的 反壟斷訴訟，法院最終認定微軟利用作業系統優勢排擠競爭對手的行為違反了托拉斯法，並判決微軟必須分拆成兩家公司。當時微軟的執行長比爾蓋茲 ( Bill Gates )，和剛回歸蘋果的執行長 Steve Jobs 簽訂了 五年合作協議，為微軟向法院爭取了一些協商時間，並成功的擺脫公司被分拆的命運。

第二次瀏覽器大戰

在 IE 統領市場後，瀏覽器的創新陷入停滯，這給了新對手挑戰的機會。2003 年，蘋果與微軟的五年協議到期，Safari 取代 Internet Explorer 成為 Mac 的預設瀏覽器，敲響了第一聲警鐘。

Netscape 在衰落前將其瀏覽器核心代碼開源，促成了 Mozilla Firefox 的誕生。2004 年 Firefox 推出，憑藉分頁瀏覽、擴充套件和更快的速度，在發布的第一年，就佔據了全球市場 10% 的份額；2009 年市占率甚至超過一度 30% ，挑戰了 IE 的霸主地位。

不讓 Firefox 專美於前，搜尋引擎龍頭 Google 於 2008 年推出 Chrome 瀏覽器，主打極簡設計、史上最快速的 V8 JavaScript 引擎、分頁沙盒技術（即使一個分頁崩潰，也不會影響整個瀏覽器），以及與 Google 搜尋服務跟 Android 作業系統的深度整合，迅速橫掃市場。

在 Safari、Firefox、Chrome 三者來勢洶洶的逼迫之下， IE 的市場份額雪崩式下滑，2012 年，Chrome 的市佔率超越 IE ，成為全球市佔第一的瀏覽器。

市場競爭所帶來的影響

在第一次瀏覽器大戰中，網景和微軟在瀏覽器開發上為了壓制對手，把加入新功能的重要性放在修正錯誤之前，導致瀏覽器消耗大量資源卻穩定性差。同時強迫使用自家專屬格式，不尊重公開網路標準，不僅讓 HTML、CSS、JavaScripts ⋯等 網路標準 (Web Standards) 的推廣與進展困難重重；也讓網路開發者必須為了不同瀏覽器開發不同版本的網頁相當困擾。

相較之下，第二次瀏覽器大戰就好得多。在 Safari、Firefox、Chrome 三足鼎立的狀況下，各家對所謂的「網路標準」比起先前更有具體的共識。Safari 的 WebKit 核心，加上 Google 的 Chromium 開源計劃 (跟 Chrome 一樣基於 Blink 核心的開源瀏覽器)，幾乎佔了市場的 95% 以上，尤其是新龍頭 Google 對追逐新技術與推動標準方面不遺餘力，讓網路環境朝著正面的方向更加蓬勃發展。
