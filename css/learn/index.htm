Day 0: CSS 10 天快速入門計劃

Learning CSS in 10 days

前言

這是一份為視覺設計師打造的 CSS 10 天快速入門計劃。以每天一到兩小時左右的學習加上練習時間，搭配簡單的案例說明，讓對前端網頁設計有興趣的視覺設計師，可以在短短的十天之內對 CSS (Cascading Style Sheet) 有基礎的認識。

學習目標

完成本課程後，你將能夠：

  • 理解 CSS 的基本語法與選擇器
  • 掌握文字排版與色彩應用
  • 理解盒模型與間距控制
  • 使用 Flexbox 和 Grid 進行版面佈局
  • 建立響應式網頁設計
  • 製作基本的 CSS 動畫效果
  • 獨立完成一個完整的響應式網頁

開發工具（免費）

  • Visual Studio Code（或其他任何IDE程式碼編輯器）
  • 網頁瀏覽器（或其他任何品牌瀏覽器皆可）

學習門檻

開始之前必備的技能：

  • 基礎 HTML 知識- 了解 HTML 標籤、元素、屬性
  • 基本電腦操作 - 能夠建立資料夾、編輯文字檔案、理解檔案結構
  • 不需要程式設計背景 - CSS 是樣式語言，不是程式語言
  • 基本的英文閱讀與理解程度
    • 理解常見的程式術語（color、background、width、height ⋯ 等）
    • 能閱讀英文技術文件（例如 MDN）更好

注意事項

CSS 是一款功能強大的樣式語言，包含了「文字排版」、「色彩配置」與「版面佈局」等核心功能。本課程專為網頁設計初學者打造，為了讓大家快速上手，精簡了許多複雜的技術細節。若您對 CSS 或前端開發感興趣，建議以這 10 天的基礎為起點，進一步探索更完整的進階技術。
