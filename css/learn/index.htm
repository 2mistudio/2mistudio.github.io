Day 0: CSS 10 天快速入門計劃

Learning CSS in 10 days

前言

這是一份為視覺設計師打造的 CSS 10 天快速入門計劃。以每天一到兩小時左右的學習加上練習時間，搭配簡單的案例說明，讓對前端網頁設計有興趣的視覺設計師，可以在短短的十天之內對 CSS (Cascading Style Sheet) 有基礎的認識。

學習目標

完成本課程後，你將能夠：

  • 理解 JavaScript 程式基本邏輯
    • 使用 let 和 const 宣告變數
    • 了解 if、else if、else 結構
    • 學會使用網頁 DOM 結構以及事件處理
  • 製作網頁互動效果（如滑鼠點擊、網頁捲動、鍵盤輸入）
    • 製作滑鼠點擊或鍵盤按鍵觸發的互動效果
    • 搭配 CSS Animation 製作出動態頁面
  • 動態操作網頁內容元素
    • 使用 JavaScript 建立網頁內容
    • 製作互動式網頁元件

開發工具（免費）

  • Chrome DevTools（或其他任何瀏覽器的開發人員工具）
  • Visual Studio Code（或其他任何IDE程式碼編輯器）

學習門檻

開始之前必備的技能：

  • 理解 HTML 與 CSS 的基本觀念
    • 知道什麼是 HTML 標籤、class、id
    • 會使用 CSS 設定顏色、間距、排版
    • 理解區塊模組（box model）的概念
  • 能獨立製作出一個簡單完整的靜態網頁
    • 能夠手刻一個簡單的 landing page
    • 或能使用 Webflow、Framer 等工具產生 HTML/CSS
  • 基本的英文閱讀與理解程度
    • 理解常見的程式術語（element、function、event 等）
    • 能閱讀英文技術文件（例如 MDN）更好

注意事項

JavaScript 是一款功能強大的程式語言，包含了運算、邏輯判斷與資料處理等核心功能。本計畫專為前端視覺設計師打造，為了讓大家快速上手，精簡了許多複雜的技術細節。若您對 JavaScript 或前端開發感興趣，建議以這 10 天的基礎為起點，進一步探索更完整的開發功能。
