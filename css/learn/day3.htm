Day 3: CSS 的文字應用

CSS Typography

一、學習目標

完成本日課程後，你將能夠：

  • 掌握字體相關屬性的使用（font-family, font-size, font-weight 等）
  • 理解文字排版屬性（line-height, letter-spacing, text-align 等）
  • 使用 font 簡寫屬性
  • 了解網頁字體的基本使用

二、Typography 文字排版基礎

Typography（字體排印）是網頁設計中最重要的元素之一。研究顯示，網頁內容的 95% 是文字，因此好的文字排版直接影響使用者體驗。

良好排版的三大要素：

  • 可讀性 (Readability)：文字容易辨識
  • 易讀性 (Legibility)：內容容易理解
  • 美觀性 (Aesthetics)：視覺上令人愉悅

三、字體屬性 (Font Properties)

1. font-family（字體家族）

指定文字使用的字體。

語法：


font-family: 字體名稱, 備用字體, 通用字體家族;

/* 範例（若字體名稱中間有空格，要用""包覆起來）*/
font-family: "Noto Sans", Arial,sans-serif;

通用字體家族：

  • serif - 襯線字體（如 Times New Roman），正式、傳統
  • sans-serif - 無襯線字體（如 Arial），現代、簡潔
  • monospace - 等寬字體（如 Courier），適合程式碼
  • cursive - 手寫體
  • fantasy - 裝飾字體

重要概念：字體堆疊 (Font Stack)

因為每個人的上網裝置安裝的字型可能不同，所以瀏覽器會按順序嘗試載入設計師指定的字體，如果第一個不可用，就嘗試下一個。最後一定要加上通用字體家族作為保險。

中文字體建議：

一般會建議以作業系統的預設系統字型為優先選擇。

  • Windows：Microsoft JhengHei (微軟正黑體)
  • macOS：PingFang TC (蘋方-繁)
  • Android：Noto Sans CJK （思源黑體）註
  • 跨平台安全選擇：sans-serif

備注：「思源黑體」是 Adobe 與 Google 共同開發的一套開源字體。同一款字體有兩種不同名稱。在 Adobe 稱之為思源黑體 Source Han Sans ，在 Google 沒有中文名稱，而英文稱為 Noto Sans （CJK 指的是其中的中文／日文 ／韓文字型）。

跨平台字體堆疊範例：


/**/
body {
  font-family: system-ui, /*W3C 標準系統字體關鍵字*/
               -apple-system, BlinkMacSystemFont, /*Mac 系統字體設定*/ 
               "Segoe UI", "Microsoft JhengHei", /*Windows 系統中文字體設定*/ 
               "PingFang TC", "Noto Sans TC", /*Android 系統中文字體設定*/ 
               Arial, /*備用字體*/
               sans-serif;/*備用字體家族*/
}

2. font-size（字體大小）

用來設定文字的大小。

常用單位：

  • px - (像素)固定大小，精確控制
  • em - 相對於父元素 (上一層) 的字體大小
  • rem - 相對於根元素 (html) 的字體大小
  • % - 相對於父元素的百分比
  • vw/vh - 相對於視窗寬度/高度

使用建議：

  • 內文：16px（瀏覽器預設的文字大小）
  • 大標題：32-48px
  • 小標題：24-28px
  • 小字：12-14px 注意

注意：部分瀏覽器（例如 Chrom系列），即使設定了小於 12px 的字級，瀏覽器仍會強制以 12px 呈現，以確保文字的可讀性。

3. font-weight（字體粗細）

用來控制文字的粗細程度（也常被稱為字重）。

設定名稱：

  • normal - 正常（等於 400）
  • bold - 粗體（等於 700）
  • lighter - 比父元素更細
  • bolder - 比父元素更粗

設定值：

  • 100（極細）到 900（極粗）
  • 常用：400（正常）、700（粗體）

注意： 不是每一種字體都支援所有的粗細設定值。

4. font-style（字體樣式）

用來設定文字的樣式。

設定值：

  • normal - 正常（預設）
  • italic - 斜體（使用字體本身的斜體版本）
  • oblique - 傾斜（強制將字體傾斜）

5. font 屬性的縮寫

只要按照適當的順序，font 其實可以一次同時設定多個字體屬性。

語法順序：


font: [font-style] [font-weight] font-size[/line-height] font-family;

/*範例*/
font: italic bold 16px/1.5 Arial, sans-serif;

注意事項：

  • font-size 和 font-family 是必須的
  • 其他屬性的設定可有可無
  • 順序很重要

四、文字排版屬性 (Text Properties)

1. color（文字顏色）

用來設定文字的顏色。

常見的文字顏色表示法：

  • 顏色名稱
  • HEX 十六進位（Hexademical Color）
  • RGB（Red、Green、Blue 的縮寫）
  • RGBA（A 為 Alpha Channel 的縮寫，可設定透明度）
  • HSL / HSLA（Hue、Saturation、Lightness）

備注： 關於顏色應用的詳細說明在下一章「 CSS 色彩空間介紹 」中會提到。

2. line-height（行高）

用來控制文字行與行之間的高度。

重要性： 行高是影響可讀性的關鍵因素。

建議值（文字大小的倍數）：

  • 內文：1.5 - 1.8
  • 標題：1.2 - 1.4
  • 程式碼：1.4 - 1.6

無單位數值的優點：


/* 推薦：無單位數值會繼承並重新計算 */
line-height: 1.6;

/* 不推薦：固定值不會隨字體大小調整 */
line-height: 24px;

3. text-align（文字對齊）

設定文字的水平對齊方式。

設定值：

  • left - 靠左對齊（預設）
  • right - 靠右對齊
  • center - 置中對齊
  • justify - 兩端對齊

備注：Left 跟 Right 是比較傳統的寫法。但世界上不是每種文字都是從左邊開始（例如阿拉伯文就是從右往左閱讀），所以比較新式的 CSS 寫法會推薦用 Start 來取代 Left，End 來取代 Right。

4. text-decoration（文字裝飾）

用來為文字添加裝飾線（通常是底線）。

設定值：

  • none - 無（常用於移除連結底線）
  • underline - 底線
  • overline - 頂線
  • line-through - 刪除線

可以組合使用：


text-decoration: underline overline;

5. text-transform（文字轉換）

用來改變文字（英文字母）的大小寫。

設定值：

  • none - 無轉換
  • uppercase - 全部大寫
  • lowercase - 全部小寫
  • capitalize - 首字母大寫

6. letter-spacing（字元間距）

用來調整字母之間的間距（適合用來調整中文橫向的字距）。

建議使用時機：

  • 標題通常需要更寬鬆的間距
  • 小字可能需要更緊密的間距以節省空間

7. word-spacing（單字間距）

調整英文單字（以空格分隔）之間的間距。

注意： 主要用於英文，中文較少使用。

8. text-indent（首行縮排）

用來設定段落第一行的文字縮排。

常見用法：


/* 中文段落縮排兩個字 */
text-indent: 2em;

五、Web Fonts 網頁字體

預設情況下，網頁只能用使用者電腦上已安裝的字體。Web Fonts 技術讓我們能使用自訂字體。

1. 使用字體服務（如 Google Fonts）

透過第三方服務載入字體，最簡單方便。

  • 前往 Google Fonts
  • 選擇字體並複製引入程式碼
  • 在 HTML <head> 中引入
  • 在 CSS 中使用

HTML:


<head>
  <!-- 引入 Google Fonts -->
  <link rel="preconnect" href="https://fonts.googleapis.com">
  <link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
</head>

CSS:


body {
  font-family: 'Roboto', sans-serif;
}

優點：

  • 使用簡單，無需下載字體檔案
  • CDN 加速，載入快速
  • Google Fonts 免費且合法授權（也有其他付費的 Web Font 服務）

缺點：

  • 依賴第三方服務，要是連線不穩會就引起網頁載入問題
  • 網頁中載入第三方程式碼，強調隱私的網站恐不適用

注意事項：

  • 載入過多字體會影響效能
  • 中文或漢字字體檔案都很大（平均 8-10MB不等），請斟酌使用
  • 建議最多不要載入超過三個網頁字體
  • 一定要提供備用字體

2. 自行託管字體檔案（@font-face）

將 Web Font 字體檔案放在自己的伺服器上，使用 @font-face 載入。

CSS:


**檔案結構：**

project/
├── fonts/
│   ├── MyFont-Regular.woff2
│   └── MyFont-Bold.woff2
├── css/
│   └── style.css
└── index.html

/* 定義字體 */
@font-face {
  font-family: 'MyFont';
  src: url('../fonts/MyFont-Regular.woff') format('woff');
  font-weight: 400;
  font-style: normal;
}

/* 使用字體 */
body {
  font-family: 'MyFont', Arial, sans-serif;
}

優點：

  • 不依賴第三方服務
  • 完全掌控載入流程
  • 可離線使用

缺點：

  • 需要自行管理字體檔案
  • 注意字體授權問題

六、常見錯誤與注意事項

錯誤 1: 字體名稱沒有加引號

錯誤示範：


/* ❌ 有空格的字體名稱要用引號 */
body {
  font-family: Microsoft JhengHei, sans-serif;
}

正確做法：


/* ✅ 有空格要加引號 */
body {
  font-family: "Microsoft JhengHei", sans-serif;
}

/* 單字字體可以不用引號 */
body {
  font-family: Arial, sans-serif;
}

錯誤 2: 忘記設定備用字體

錯誤示範：


/* ❌ 沒有備用字體 */
body {
  font-family: "特殊字體";
}

正確做法：


/* ✅ 提供備用方案，最後加上通用字體家族 */
body {
  font-family: "特殊字體", Arial, sans-serif;
}

錯誤 3: 行高設定不當

錯誤示範：


/* ❌ 行高太小，文字擁擠 */
p {
  line-height: 1.0;
}

/* ❌ 使用固定值，不會隨字體大小調整 */
p {
  font-size: 16px;
  line-height: 24px;
}

正確做法：


/* ✅ 使用無單位數值 */
p {
  line-height: 1.6;  /* 推薦範圍：1.5-1.8 */
}

錯誤 4: font 簡寫順序錯誤

錯誤示範：


/* ❌ 順序不對 */
p {
  font: Arial 16px bold;
}

正確做法：


/* ✅ 正確順序：style weight size/line-height family */
p {
  font: bold 16px/1.6 Arial, sans-serif;
}

/* 或分開寫 */
p {
  font-size: 16px;
  font-weight: bold;
  font-family: Arial, sans-serif;
}

七、重點整理

字體屬性：

  • font-family - 設定字體，要提供備用方案
  • font-size - 字體大小（建議用 px 或 rem）
  • font-weight - 字體粗細（400=normal, 700=bold）
  • font-style - 斜體效果（italic）
  • font - 簡寫屬性，順序：style weight size/line-height family

文字排版：

  • line-height - 行高，推薦使用無單位數值（1.5-1.8）
  • text-align - 對齊方式（left, center, right, justify）
  • text-decoration - 文字裝飾線（none, underline, line-through）
  • text-transform - 大小寫轉換（uppercase, lowercase, capitalize）
  • letter-spacing - 字元間距
  • text-indent - 首行縮排

Web Fonts：

  • 使用 Google Fonts 載入自訂字體
  • 在 HTML 引入，在 CSS 使用
  • 注意效能，限制字體數量
  • 一定要提供備用字體

最佳實踐：

  • 內文字體大小至少 16px
  • 最小字體不得小於 12px
  • 行高建議 1.5-1.8
  • 字體名稱有空格要加引號
  • 使用字體堆疊提供備用方案
