Day 6: Flexbox 彈性佈局

CSS Display and Position

本單元算是 「CSS 網頁排版」相關的第一個章節，尤其是 CSS Position 部分，雖然是相當早期的 CSS 技術，但是因為在排版上相當實用，至今還是非常重要常見的排版技術。

一、學習目標

完成本日課程後，你將能夠：

  • 理解 display 屬性的作用與常用值
  • 掌握 block、inline、inline-block 的差異
  • 了解 position 屬性的五種定位方式
  • 使用 top、right、bottom、left 控制元素位置
  • 理解 z-index 的層疊順序控制

二、Display 屬性

Display 屬性決定元素如何顯示及與其他元素的排列方式，有許多種不同的性設定：

  • display: block （顯示為區塊元素）
  • display: inline （顯示為行內元素）
  • display: inline-block （顯示為行內區塊）
  • display: none （不顯示）
  • display: flex （會在後面的章節介紹）
  • display: grid （會在後面的章節介紹）

1. block 區塊元素

HTML 元素有許多預設的屬性，區塊元素就是其中之一。

特性：

  • 獨佔一行空間，後面的元素會自動換行
  • 可以設定寬度和高度
  • 預設寬度為父元素的 100%

預設為 block 的元素： <div>, <p>, <h1>-<h6>, <ul>, <ol>, <li>, <section>, <header>, <footer>⋯等。

2. inline 行內元素

行內元素元素也是 HTML 元素許多預設的屬性之一。

特性：

  • 不會換行，與其他元素在同一行
  • 不能設定寬度和高度
  • 寬度由內容決定
  • 垂直方向的 margin 和 padding 無效

預設為 inline 的元素： <span>, <a>, <strong>, <em>, <img>, <input>⋯等。

3. inline-block 行內區塊

特性：

  • 不會換行，與其他元素在同一行
  • 可以設定寬度和高度
  • 結合了 inline 和 block 的優點

特性：

  • 導覽列按鈕
  • 圖片排列
  • 標籤或徽章

4. none 隱藏元素

用來讓指定 HTML 元素從網頁中完全消失。

特性：

  • 元素完全不顯示
  • 不佔據空間

與 visibility: hidden 的差異：

  • display: none - 不顯示，不佔空間
  • visibility: hidden - 不顯示，但仍佔空間

Display 屬性比較表

屬性 block inline inline-block
換行 是 否 否
設定寬高 可以 不可以 可以
垂直 margin/padding 有效 無效 有效
預設寬度 100% 內容決定 內容決定

三、Position 屬性

position 屬性決定元素的位置，一共有五種定位設定方式：

  • position: static（靜態定位，預設值）
  • position: relative（相對定位）
  • position: absolute（絕對定位）
  • position: fixed（固定定位）
  • position: sticky（黏著定位）

1. static（靜態定位，預設值）

特性：

  • 元素按照正常文檔流（從上到下、由左而右）排列
  • top、right、bottom、left、z-index 無效

2. relative（相對定位）

特性：

  • 相對於元素原本的位置進行偏移
  • 原本所在的空間仍然保留
  • 不影響其他元素的位置

使用場景：

  • 微調元素位置
  • 作為 absolute 定位的參考點（父元素）

3. absolute（絕對定位）

特性：

  • 相對於最近的 已定位父元素（非 static）進行定位
  • 如果沒有已定位的父元素，則相對於 <body> 標籤位置（也就是網頁頁面的左上角）
  • 設定絕對定位的元素會脫離正常文檔流，不佔空間
  • 其他元素會忽略它的存在

使用場景：

  • 彈出視窗
  • 下拉選單
  • 浮動按鈕
  • 徽章或標記

4. fixed（固定定位）

特性：

  • 相對於瀏覽器視窗進行定位
  • 脫離正常文檔流
  • 捲動頁面時位置不變

使用場景：

  • 固定導覽列
  • 回到頂部按鈕
  • 浮動客服視窗

5. sticky（黏著定位）

特性：

  • 結合 relative 和
  • 在捲動到指定位置前表現為 relative
  • 捲動到指定位置後表現為 fixed
  • 需要設定 top、bottom 等值

使用場景：

  • 頁首選單
  • 表格標題
  • 側邊欄
  • 區塊標題

四、定位偏移屬性

用來配合 position 使用（static 除外）：


top: 值;      /* 距離頂部 */
right: 值;    /* 距離右側 */
bottom: 值;   /* 距離底部 */
left: 值;     /* 距離左側 */

五、z-index（層疊順序）

可以想像成 Photoshop 的圖層概念，用來控制被定位的 HTML 元素的堆疊順序（只對已設定定位的元素有效）。

特性：

  • 數值越大，越在上層
  • 預設值為 auto（等於 0）
  • 可以使用負值

七、重點整理

Box Model 盒模型組成：

  • Content（內容）→ Padding（內邊距）→ Border（邊框）→ Margin（外邊距）

Margin & Padding 間距屬性：

  • margin - 外邊距，可用負值，margin: 0 auto 可水平置中 padding - 內
  • font-size - 字體大小（建議用 px 或 rem）
  • padding - 內邊距，不可用負值
  • 設定方式：四個值 / 兩個值 / 一個值 / 單獨設定

Border 邊框：

  • border: 寬度 樣式 顏色
  • 樣式：solid、dashed、dotted、double、none
  • border-radius - 圓角，50% 可製作圓形

Width & Height 寬高尺寸：

  • width、height - 寬度和高度
  • max-width、min-width - 最大/最小寬度
  • max-height、min-height - 最大/最小高度

Box-sizing：

  • content-box - 預設，寬度不包含 padding 和 border
  • border-box - 推薦，寬度包含 padding 和 border
  • 建議全域設定為 border-box

Background 背景：

  • background-color - 背景顏色
  • background-image - 背景圖片
  • background-size - 圖片尺寸（cover / contain）
  • background-position - 圖片位置
  • background-repeat - 是否重複

重要觀念：

  • 垂直 margin 會疊加（取較大值）
  • Padding 不能用負值，Margin 可以
  • 百分比高度需要父元素有明確高度
  • 推薦使用 box-sizing: border-box
