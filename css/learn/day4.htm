Day 4: Box Model 與間距控制

Box Model, Margin and Padding

這個單元可以說是 「CSS 最重要的概念」之一。初學者通常很難將 CSS 的 Box Model 一次弄清楚。只要能夠弄清楚區塊與間距的邏輯，後續遇到的問題會少非常多。所以本單元雖然段落章節比較多，但是內容不算長，非常值得多花點時間理解跟練習。

一、學習目標

完成本日課程後，你將能夠：

  • 理解 CSS 盒子模型（區塊）的組成與運作原理
  • 掌握 margin、padding、border 的使用
  • 了解 box-sizing 屬性的作用
  • 控制元素的寬度和高度
  • 設定元素的背景樣式

二、CSS Box Model 盒子模型

每個 HTML 元素都可被看作是一個「方塊盒子」，由內到外包含四個部分：

Content（內容）→ Padding（內邊距）→ Border（邊框）→ Margin（外邊距）

視覺示意：

三、Margin 外邊距

HTML 元素與外圍其他元素之間的距離。

基本語法：


margin: 值;

四個方向的設定：


/* 四個值（上 右 下 左，順時針） */
margin: 10px 20px 30px 40px;

/* 兩個值（上下 左右） */
margin: 10px 20px;

/* 一個值（四邊相同） */
margin: 10px;

/* 單獨設定 */
margin-top: 10px;
margin-right: 20px;
margin-bottom: 30px;
margin-left: 40px;

水平置中技巧：


.center {
  width: 300px;
  margin: 0 auto;  /* 上下 0，左右自動 */
}

負值應用： Margin 可以使用負值，讓元素重疊或突破容器。


.overlap {
  margin-top: -20px;  /* 向上移動 20px */
}

四、Padding 內邊距

內容與邊框之間的距離。

語法：與 margin 相同。


/* 四個值 */
padding: 10px 20px 30px 40px;

/* 兩個值 */
padding: 10px 20px;

/* 一個值 */
padding: 10px;

/* 單獨設定 */
padding-top: 10px;
padding-right: 20px;
padding-bottom: 30px;
padding-left: 40px;

注意： Padding 不能使用負值。

五、Border 邊框

HTML 元素的邊框樣式。

基本語法：


border: 寬度 樣式 顏色;

基本語法：

  • solid - 實線
  • dashed - 虛線
  • dotted - 點線
  • double - 雙線
  • none - 無邊框

完整設定：


/* 簡寫 */
border: 2px solid #333;

/* 分開設定 */
border-width: 2px;
border-style: solid;
border-color: #333;

/* 單邊設定 */
border-top: 3px dashed blue;
border-bottom: 1px solid gray;

圓角邊框：


border-radius: 10px;  /* 四角相同 */
border-radius: 10px 20px 30px 40px;  /* 左上 右上 右下 左下 */

/* 圓形 */
border-radius: 50%;

六、Width ／ Height 寬度與高度

基本語法：


width: 值;
height: 值;

常用單位：

  • px - 像素
  • % - 百分比（相對於上一層 HTML 父元素）
  • auto - 自動（預設值）

最大/最小值：用在響應式設計，防止網頁內容元件過大或過小


max-width: 1200px;   /* 最大寬度 */
min-width: 300px;    /* 最小寬度 */
max-height: 500px;   /* 最大高度 */
min-height: 200px;   /* 最小高度 */

⚠️ 重要：寬度與高度的 Box-sizing 屬性

決定寬度和高度的計算方式。

1. content-box（預設值）

  • 總寬度 = width + padding + border

2. border-box（推薦用法）

  • 總寬度 = width（padding 和 border 包含在內）

範例對比：


/* content-box（預設） */
.box1 {
  width: 300px;
  padding: 20px;
  border: 5px solid black;
  /* 實際寬度 = 300 + 40 + 10 = 350px */
}

/* border-box（推薦用法） */
.box2 {
  box-sizing: border-box;
  width: 300px;
  padding: 20px;
  border: 5px solid black;
  /* 實際寬度 = 300px */
}

全域設定（推薦）：通常會寫在 CSS 一開始的地方，讓它影響整個頁面


* {
  box-sizing: border-box;
}

注意事項：

除非是明確需要高度設定的內容，一般實作上會盡量不去設定高度限制，讓內容多寡自動決定區塊的高度，以維持版面的彈性。

七、Background 背景

背景語法：


background-color: #f0f0f0;         /*背景顏色*/
background-image: url('image.jpg');/*背景圖片*/
background-size: cover;            /* cover 或 contain */
background-position: center;       /* 位置 */
background-repeat: no-repeat;      /* 是否重複(預設背景會重複) */

背景語法簡寫：


/* 排列順序：color image repeat position/size */
background: #f0f0f0 url('image.jpg') no-repeat center/cover;

背景尺寸完整使用說明：


/* 關鍵字 */
background-size: cover;    /* 填滿容器，可能裁切圖片 */
background-size: contain;  /* 完整顯示圖片，可能留白 */

/* 固定尺寸 */
background-size: 200px 100px;  /* 寬 200px，高 100px */
background-size: 100px;        /* 寬 100px，高度自動等比例 */

/* 百分比 */
background-size: 100% 50%;  /* 寬度 100%，高度 50% */
background-size: 50%;       /* 寬度 50%，高度自動 */

/* 自動 */
background-size: auto;  /* 圖片原始尺寸 */

背景位置完整使用說明：


/* 關鍵字 */
background-position: center;        /* 置中 */
background-position: top;           /* 頂部 */
background-position: bottom;        /* 底部 */
background-position: left;          /* 左側 */
background-position: right;         /* 右側 */

/* 組合關鍵字 */
background-position: top left;      /* 左上角 */
background-position: top right;     /* 右上角 */
background-position: bottom left;   /* 左下角 */
background-position: bottom right;  /* 右下角 */
background-position: center top;    /* 水平置中，垂直靠上 */

/* 精確數值（X軸 Y軸） */
background-position: 20px 30px;     /* 距左 20px，距上 30px */
background-position: 50px 0;        /* 距左 50px，頂部 */

/* 百分比（X% Y%） */
background-position: 50% 50%;       /* 置中（同 center） */
background-position: 0% 0%;         /* 左上角（同 top left） */
background-position: 100% 100%;     /* 右下角（同 bottom right） */

/* 混合使用 */
background-position: center 20px;   /* 水平置中，距上 20px */
background-position: 80% bottom;    /* 水平 80%，底部 */

背景重複完整使用說明：


background-repeat: no-repeat;  /* 不重複 */
background-repeat: repeat;     /* 重複（預設） */
background-repeat: repeat-x;   /* 只水平重複 */
background-repeat: repeat-y;   /* 只垂直重複 */

八、常見錯誤與注意事項

錯誤 1: 忘記設定 box-sizing

問題示範：


/* ❌ 沒有設定 box-sizing */
.box {
  width: 300px;
  padding: 20px; /*上下左右的內邊距*/
  border: 5px solid black; /*上下左右的邊框*/
  /* 實際寬度變成 300 + 20 + 20 + 5 + 5 = 350px */
}

建議做法：


/* ✅ 全域設定 border-box */
* {
  box-sizing: border-box;
}

.box {
  width: 300px;
  padding: 20px;
  border: 5px solid black;
  /* 實際寬度就是 300px */
}

錯誤 2: Margin 疊加問題

垂直 Margin 會疊加（兩組設定會取較大值）：


.box1 {
  margin-bottom: 20px;
}

.box2 {
  margin-top: 30px;
}
/* 結果兩個元素之間的距離是 30px，不是 50px */

解決方式：

  • 了解這是正常行為
  • 只設定單方向的 margin（如只設定 margin-bottom）
  • 如想要一步理解這個現象，可延伸閱讀關於 margin collapse 問題。

錯誤 3: Padding 使用負值

錯誤示範：


/* ❌ Padding 不能使用負值 */
.box {
  padding: -10px;  /* 無效 */
}

只有 margin 可以使用負值，Padding 不行。

錯誤 4: 百分比高度問題

問題示範：


/* ❌ 父元素沒有明確高度時，百分比高度無效 */
.parent {
  /* 沒有設定 height */
}

.child {
  height: 50%;  /* 無效 */
}

正確做法：


/* ✅ 父元素要有明確高度 */
.parent {
  height: 500px;
}

.child {
  height: 50%;  /* 有效，250px */
}

高度設定在網頁中是一個特別的存在。

  • 網頁中 HTML 區塊元素預設的寬度是 100%
  • 網頁中 HTML 區塊元素預設的高度卻是 0
  • 高度設定為百分比時，父元素上必須要有高度設定才有作用
  • 網頁中最高層的父元素為 <html>，其次為 <body>，因此常見的做法是在這些標籤上擇一設定 height: 100%，就可以解決這個問題

錯誤 5: Border 語法順序錯誤

錯誤示範：


/* ❌ 順序不對 */
.box {
  border: blue 2px solid;
}

正確做法：


/* ✅ 正確順序：寬度 樣式 顏色 */
.box {
  border: 2px solid blue;
}

/* 順序可以省略，但建議按慣例 */
.box {
  border: solid 2px blue;  /* 也可以，但不推薦 */
}

九、重點整理

Box Model 盒模型組成：

  • Content（內容）→ Padding（內邊距）→ Border（邊框）→ Margin（外邊距）

Margin & Padding 間距屬性：

  • margin - 外邊距，可用負值，margin: 0 auto 可水平置中 padding - 內
  • font-size - 字體大小（建議用 px 或 rem）
  • padding - 內邊距，不可用負值
  • 設定方式：四個值 / 兩個值 / 一個值 / 單獨設定

Border 邊框：

  • border: 寬度 樣式 顏色
  • 樣式：solid、dashed、dotted、double、none
  • border-radius - 圓角，50% 可製作圓形

Width & Height 寬高尺寸：

  • width、height - 寬度和高度
  • max-width、min-width - 最大/最小寬度
  • max-height、min-height - 最大/最小高度

Box-sizing：

  • content-box - 預設，寬度不包含 padding 和 border
  • border-box - 推薦，寬度包含 padding 和 border
  • 建議全域設定為 border-box

Background 背景：

  • background-color - 背景顏色
  • background-image - 背景圖片
  • background-size - 圖片尺寸（cover / contain）
  • background-position - 圖片位置
  • background-repeat - 是否重複

重要觀念：

  • 垂直 margin 會疊加（取較大值）
  • Padding 不能用負值，Margin 可以
  • 百分比高度需要父元素有明確高度
  • 推薦使用 box-sizing: border-box
