Day 10: 事件處理基礎

Event Handlers Basic

一、學習目標

完成本日學習後，你應該能夠：

  1. 理解事件（Event）的概念
  2. 使用 addEventListener 監聽各種事件
  3. 處理點擊、滑鼠移入移出、鍵盤輸入等常見事件
  4. 理解事件物件（Event Object）的使用
  5. 製作豐富的互動效果

二、核心概念解釋

什麼是事件？

事件（Event）是「使用者與網頁互動時發生的事情」。每當使用者做了某個動作（點擊、移動滑鼠、按鍵盤、滾動頁面...），瀏覽器就會產生一個對應的事件。

JavaScript 透過「事件監聽器」（Event Listener）來「監聽」這些事件，當事件發生時，執行對應的程式碼。

事件監聽器的運作方式


element.addEventListener("事件類型", function() {
    // 當事件發生時，要執行的程式碼
});

這就像在 Figma 中設定 Prototype：

  • 選取元素 → element
  • 設定觸發條件 → "事件類型"（如 click、hover）
  • 定義動作 → function() { ... }（執行什麼）

常見的事件類型

滑鼠事件：

  • click → 點擊
  • dblclick → 雙擊
  • mouseenter → 滑鼠移入
  • mouseleave → 滑鼠移出
  • mousemove → 滑鼠移動

鍵盤事件：

  • keydown → 按下鍵盤按鍵
  • keyup → 放開鍵盤按鍵
  • keypress → 按下字元鍵（不推薦，已過時）

表單事件：

  • input → 輸入框內容改變（即時）
  • change → 輸入框失去焦點且內容改變
  • focus → 輸入框獲得焦點
  • blur → 輸入框失去焦點
  • submit → 表單提交

其他事件：

  • scroll → 滾動頁面
  • resize → 視窗大小改變
  • load → 頁面載入完成

事件物件（Event Object）

當事件發生時，瀏覽器會自動建立一個「事件物件」，包含了事件的詳細資訊：


element.addEventListener("click", function(event) {
    console.log(event.target);      // 被點擊的元素
    console.log(event.clientX);     // 滑鼠 X 座標
    console.log(event.clientY);     // 滑鼠 Y 座標
    console.log(event.key);         // 按下的按鍵（鍵盤事件）
});

就像在設計軟體中，你可以知道「使用者點擊了哪個圖層」、「滑鼠在哪個位置」。

範例一：基本點擊事件


// 建立按鈕
let btn = document.createElement("button");
btn.textContent = "點我！";
document.querySelector(".container").appendChild(btn);

// 監聽點擊事件
btn.addEventListener("click", function() {
    alert("按鈕被點擊了！");
    console.log("Click 事件觸發");
});

執行後，點擊按鈕會跳出警告視窗。

說明： addEventListener 的第一個參數是事件類型（"click"），第二個參數是事件處理函式（當事件發生時要執行的程式碼）。

範例二：改變元素樣式（點擊切換顏色）


let box = document.createElement("div");
box.className = "box";
box.textContent = "點我改變顏色";
document.querySelector(".container").appendChild(box);

let colors = ["#667eea", "#f093fb", "#4facfe", "#43e97b", "#fa709a"];
let currentIndex = 0;

box.addEventListener("click", function() {
    currentIndex++;
    if (currentIndex >= colors.length) {
        currentIndex = 0;
    }
    box.style.backgroundColor = colors[currentIndex];
    console.log("顏色已改變為：" + colors[currentIndex]);
});

執行後，每次點擊方塊會循環切換顏色。

說明： 這展示了如何在事件處理函式中修改元素樣式，創造互動效果。

範例三：滑鼠移入/移出事件


let hoverBox = document.createElement("div");
hoverBox.className = "box";
hoverBox.textContent = "滑鼠移到我身上";
hoverBox.style.backgroundColor = "#6c757d";
document.querySelector(".container").appendChild(hoverBox);

// 滑鼠移入
hoverBox.addEventListener("mouseenter", function() {
    hoverBox.textContent = "😊 歡迎！";
    hoverBox.style.backgroundColor = "#28a745";
    hoverBox.style.transform = "scale(1.2)";
});

// 滑鼠移出
hoverBox.addEventListener("mouseleave", function() {
    hoverBox.textContent = "👋 再見！";
    hoverBox.style.backgroundColor = "#dc3545";
    hoverBox.style.transform = "scale(1)";
});

執行後，滑鼠移入移出方塊會有不同的視覺反饋。

說明： mouseenter 和 mouseleave 常用於製作 hover 效果，比純 CSS 更靈活。

範例四：鍵盤事件（即時顯示按鍵）


let section = document.createElement("div");
section.className = "section";
section.innerHTML = `
    
⌨️ 鍵盤事件

    
按下任意鍵盤按鍵...

    
等待按鍵...

`;
document.querySelector(".container").appendChild(section);

let keyOutput = document.querySelector("#keyOutput");

document.addEventListener("keydown", function(event) {
    keyOutput.innerHTML = `
        按鍵：${event.key}

        按鍵代碼：${event.code}

        是否按住 Shift：${event.shiftKey ? "是" : "否"}
    `;
    
    // 特殊按鍵處理
    if (event.key === "Enter") {
        keyOutput.style.backgroundColor = "#d4edda";
    } else if (event.key === "Escape") {
        keyOutput.style.backgroundColor = "#f8d7da";
    } else {
        keyOutput.style.backgroundColor = "#d1ecf1";
    }
});

執行後，按下鍵盤任意按鍵會即時顯示按鍵資訊。

說明： 事件物件的 event.key 可以取得按下的按鍵，event.code 是按鍵的物理位置代碼。

範例五：表單輸入事件（即時字數統計）


let formSection = document.createElement("div");
formSection.className = "section";
formSection.innerHTML = `
    
📝 即時字數統計

    
    
字數：0

`;
document.querySelector(".container").appendChild(formSection);

let textInput = document.querySelector("#textInput");
let charCount = document.querySelector("#charCount");

textInput.addEventListener("input", function(event) {
    let length = event.target.value.length;
    charCount.textContent = `字數：${length}`;
    
    // 根據字數改變顏色
    if (length === 0) {
        charCount.style.color = "#6c757d";
    } else if (length < 10) {
        charCount.style.color = "#28a745";
    } else if (length < 20) {
        charCount.style.color = "#ffc107";
    } else {
        charCount.style.color = "#dc3545";
    }
});

執行後，輸入文字時會即時顯示字數，並根據長度改變顏色。

說明： input 事件會在輸入框內容改變時立即觸發，event.target.value 可以取得輸入框的值。

範例六：事件物件的使用（點擊顯示座標）


let coordSection = document.createElement("div");
coordSection.className = "section";
coordSection.innerHTML = `
    
🎯 點擊座標追蹤

    

        點擊我！
    

    
點擊方塊查看座標...

`;
document.querySelector(".container").appendChild(coordSection);

let clickBox = document.querySelector("#clickBox");
let coordOutput = document.querySelector("#coordOutput");

clickBox.addEventListener("click", function(event) {
    coordOutput.innerHTML = `
        滑鼠位置（相對於視窗）：

        X: ${event.clientX}px, Y: ${event.clientY}px


        滑鼠位置（相對於元素）：

        X: ${event.offsetX}px, Y: ${event.offsetY}px
    `;
    
    // 在點擊位置產生一個小圓點
    let dot = document.createElement("div");
    dot.style.position = "absolute";
    dot.style.left = event.clientX + "px";
    dot.style.top = event.clientY + "px";
    dot.style.width = "10px";
    dot.style.height = "10px";
    dot.style.background = "#ff6b6b";
    dot.style.borderRadius = "50%";
    dot.style.pointerEvents = "none";
    document.body.appendChild(dot);
    
    // 1 秒後移除圓點
    setTimeout(function() {
        dot.remove();
    }, 1000);
});

執行後，點擊方塊會顯示座標，並在點擊位置出現一個短暫的紅點。

說明： 事件物件包含豐富的資訊，clientX/clientY 是相對於視窗的座標，offsetX/offsetY 是相對於元素的座標。

範例七：完整範例（計數器）


let counterSection = document.createElement("div");
counterSection.className = "section";
counterSection.innerHTML = `
    
🔢 互動式計數器

    
0

    

        
        
        
    

`;
document.querySelector(".container").appendChild(counterSection);

let counter = document.querySelector("#counter");
let decreaseBtn = document.querySelector("#decreaseBtn");
let resetBtn = document.querySelector("#resetBtn");
let increaseBtn = document.querySelector("#increaseBtn");

let count = 0;

// 更新顯示的函式
function updateDisplay() {
    counter.textContent = count;
    
    // 根據數值改變顏色
    if (count > 0) {
        counter.style.color = "#28a745";
    } else if (count < 0) {
        counter.style.color = "#dc3545";
    } else {
        counter.style.color = "#667eea";
    }
}

// 增加
increaseBtn.addEventListener("click", function() {
    count++;
    updateDisplay();
    console.log("增加，目前：" + count);
});

// 減少
decreaseBtn.addEventListener("click", function() {
    count--;
    updateDisplay();
    console.log("減少，目前：" + count);
});

// 重置
resetBtn.addEventListener("click", function() {
    count = 0;
    updateDisplay();
    console.log("重置");
});

// 鍵盤快捷鍵
document.addEventListener("keydown", function(event) {
    if (event.key === "ArrowUp") {
        count++;
        updateDisplay();
    } else if (event.key === "ArrowDown") {
        count--;
        updateDisplay();
    } else if (event.key === "r" || event.key === "R") {
        count = 0;
        updateDisplay();
    }
});

執行後，可以用按鈕或鍵盤方向鍵控制計數器。

說明： 這個完整範例展示了如何結合多個事件監聽器，創造豐富的互動體驗。

四、常見錯誤與注意事項

錯誤一：忘記 function 關鍵字或括號

錯誤範例：


// 錯誤：缺少 function
btn.addEventListener("click", {
    console.log("點擊");
});

// 錯誤：多了括號（會立即執行）
btn.addEventListener("click", myFunction());

正確寫法：


// 方法一：匿名函式
btn.addEventListener("click", function() {
    console.log("點擊");
});

// 方法二：具名函式（不加括號）
function handleClick() {
    console.log("點擊");
}
btn.addEventListener("click", handleClick);

說明： 事件監聽器需要的是「函式本身」，不是「函式執行的結果」。加上括號 () 會立即執行函式。

錯誤二：事件類型拼錯字

錯誤範例：


btn.addEventListener("clik", function() {  // 拼錯：clik（應該是 click）
    console.log("不會執行");
});

btn.addEventListener("Click", function() {  // click 大小寫錯誤
    console.log("不會執行");
});

正確寫法：


btn.addEventListener("click", function() {  // 正確：全小寫
    console.log("會執行");
});

常見事件類型（記得全部都是小寫）：

  • click、dblclick
  • mouseenter、mouseleave
  • keydown、keyup
  • input、change、submit

錯誤三：在不存在的元素上綁定事件

錯誤範例：


// 元素還不存在
let btn = document.querySelector("#myBtn");
btn.addEventListener("click", function() {  // 錯誤：btn 是 null
    console.log("點擊");
});

// 之後才建立元素
let button = document.createElement("button");
button.id = "myBtn";

正確順序：


// 先建立元素
let button = document.createElement("button");
button.id = "myBtn";
button.textContent = "點我";

// 再綁定事件
button.addEventListener("click", function() {
    console.log("點擊");
});

// 最後插入 DOM
document.body.appendChild(button);

錯誤四：重複綁定相同事件

問題範例：


let btn = document.querySelector("#btn");

// 在迴圈中重複綁定（不好的做法）
for (let i = 0; i < 5; i++) {
    btn.addEventListener("click", function() {
        console.log("點擊 " + i);
    });
}

// 點一次按鈕會執行 5 次！

說明： 每次 addEventListener 都會新增一個監聽器，不會覆蓋原有的。如果不小心重複綁定，會導致事件被觸發多次。

解決方法： 只綁定一次，或使用具名函式便於移除：


function handleClick() {
    console.log("點擊");
}

btn.addEventListener("click", handleClick);

// 需要時可以移除
// btn.removeEventListener("click", handleClick);

錯誤五：this 的指向問題

容易混淆的範例：


let btn = document.querySelector("#btn");

btn.addEventListener("click", function() {
    console.log(this);  // this 指向 btn 元素
    this.style.backgroundColor = "red";  // 可以用
});

// 使用箭頭函式
btn.addEventListener("click", () => {
    console.log(this);  // this 指向 window（通常不是你要的）
    this.style.backgroundColor = "red";  // 無法正常運作
});

說明： 在一般函式中，this 指向觸發事件的元素。但在箭頭函式中，this 繼承自外層作用域。

建議： 如果需要使用 this 指向元素，使用一般函式。或使用 event.target：


btn.addEventListener("click", (event) => {
    event.target.style.backgroundColor = "red";  // 推薦
});

五、重點整理

  • 事件監聽器語法：element.addEventListener("事件類型", function() { ... })
  • 常用事件：
    • 滑鼠：click、dblclick、mouseenter、mouseleave
    • 鍵盤：keydown、keyup
    • 表單：input、change、focus、blur、submit
  • 事件物件：事件處理函式的第一個參數，包含事件的詳細資訊（座標、按鍵、目標元素等）。
  • 綁定時機：元素必須存在才能綁定事件，動態建立的元素需要在建立後立即綁定。
  • 避免重複綁定：同一個元素上不要重複綁定相同的事件監聽器。
