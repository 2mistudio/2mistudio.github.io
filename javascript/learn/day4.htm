Day 4: 條件判斷 ( if / else )

Conditional Operator ( if/else )

一、學習目標

完成本日學習後，你應該能夠：

  1. 理解條件判斷的概念與應用場景
  2. 正確使用 if、else if、else 語句
  3. 撰寫巢狀條件判斷結構
  4. 使用條件判斷解決實際問題
  5. 理解真值（truthy）與假值（falsy）的概念

二、核心概念解釋

什麼是條件判斷？

條件判斷是程式設計中控制執行流程的基本工具。在現實生活中，我們經常需要根據不同情況做出不同決策，例如「如果下雨就帶傘，否則不帶」。程式中的條件判斷就是這個邏輯的實現，讓程式能夠根據特定條件執行不同的程式碼區塊。

條件判斷的核心是「Boolean 布林邏輯」。條件表達式會被評估為 true（真）或 false（假），程式會根據這個結果決定要執行哪一段程式碼。這種能力讓程式從單純的線性執行變得更加智慧與靈活。

if 語句的基本結構

最基本的條件判斷是 if 語句。語法為：if (條件) { 程式碼 }。當條件為 true 時，大括號內的程式碼會被執行；條件為 false 時則會跳過。

當需要處理「條件不成立」的情況時，可以加上 else 子句。if...else 結構確保兩個分支中必定會有一個被執行，就像岔路一樣，必須選擇其中一條路走。

多重條件判斷

當需要檢查多個條件時，可以使用 else if。這種結構會依序檢查每個條件，一旦有條件成立就執行對應的程式碼並跳出整個判斷結構。這在成績評等、會員等級判斷等場景中特別實用。

真值與假值

JavaScript 中，不只有 true 和 false 可以用於條件判斷。所有的值都可以被轉換為布林值。會被視為 false 的值（假值）包括：false、0、空字串（""）、null、undefined、NaN。除此之外的值都被視為 true（真值），包括非空字串、非零數字、物件、陣列等。

這個特性讓條件判斷更簡潔，但也需要小心使用，避免非預期的判斷結果。

三、語法與使用方式

範例一：基本 if 語句


let temperature = 35;

if (temperature > 30) {
  console.log("天氣很熱，記得多喝水！");
}

** 執行結果：**
天氣很熱，記得多喝水！

說明： 當條件 temperature > 30 為 true 時，執行大括號內的程式碼。如果 temperature 是 25，條件為 false，程式碼不會執行，也不會有任何輸出。

範例二：if...else 結構


let age = 16;

if (age >= 18) {
  console.log("已成年，可以投票");
} else {
  console.log("未成年，不能投票");
}

** 執行結果：**
未成年，不能投票

說明： age 是 16，小於 18，條件為 false，所以執行 else 區塊。if 和 else 兩個區塊必定有一個會執行。

範例三：else if 多重條件


let score = 75;

if (score >= 90) {
  console.log("成績：A");
} else if (score >= 80) {
  console.log("成績：B");
} else if (score >= 70) {
  console.log("成績：C");
} else if (score >= 60) {
  console.log("成績：D");
} else {
  console.log("成績：F（不及格）");
}

** 執行結果：**
成績：C

說明： 程式從上到下依序檢查條件。score 是 75，第一、二個條件都不符合，第三個條件 score >= 70 成立，執行對應程式碼後，跳出整個判斷結構，後續條件不再檢查。

範例四：巢狀條件判斷


let isLoggedIn = true;
let isAdmin = false;

if (isLoggedIn) {
  if (isAdmin) {
    console.log("歡迎，管理員！");
  } else {
    console.log("歡迎，一般使用者！");
  }
} else {
  console.log("請先登入");
}

** 執行結果：**
歡迎，一般使用者！

說明： 巢狀條件判斷是在 if 區塊內再包含 if 語句。外層條件成立後，才會檢查內層條件。這種結構適合處理「條件之間有依賴關係」的情況。

範例五：真值與假值判斷


let userName = "";

if (userName) {
  console.log("歡迎，" + userName);
} else {
  console.log("請輸入使用者名稱");
}

** 執行結果：**
請輸入使用者名稱

說明： 空字串是假值，所以 if (userName) 的條件為 false，執行 else 區塊。這種簡潔寫法常用於檢查變數是否有值。

範例六：邏輯運算子結合條件判斷


let age = 25;
let hasLicense = true;

if (age >= 18 && hasLicense) {
  console.log("可以開車");
} else {
  console.log("不能開車");
}

** 執行結果：**
可以開車

說明： 使用 && 組合多個條件，兩個條件都必須為 true，整個條件才為 true。

四、常見錯誤與注意事項

錯誤一：使用賦值而非比較

錯誤範例：


let x = 5;

if (x = 10) {  // 錯誤：這是賦值，不是比較
  console.log("x 等於 10");
}

問題： = 是賦值運算子，== 或 === 才是比較運算子。上面的程式碼會將 x 的值改為 10，並且因為 10 是真值，條件永遠成立。

正確寫法：


let x = 5;

if (x === 10) {  // 正確：使用比較運算子
  console.log("x 等於 10");
}

錯誤二：忘記大括號

不良範例：


let score = 50;

if (score >= 60)
  console.log("及格");
  console.log("恭喜你！");  // 這行總是會執行

問題： 沒有大括號時，if 只控制緊接著的第一行。第二個 console.log 不受 if 控制，無論條件如何都會執行。

正確寫法：


let score = 50;

if (score >= 60) {
  console.log("及格");
  console.log("恭喜你！");
}

最佳實踐： 即使只有一行程式碼，也建議加上大括號，提高可讀性並避免錯誤。

錯誤三：條件順序錯誤

錯誤範例：


let score = 95;

if (score >= 60) {
  console.log("及格");
} else if (score >= 90) {  // 這個條件永遠不會被檢查到
  console.log("優秀");
}

** 執行結果：**
及格

問題： 條件由上而下依序檢查，一旦某個條件成立就不再檢查後續條件。95 >= 60 成立，執行後就結束，永遠不會檢查到 >= 90。

正確寫法：


let score = 95;

if (score >= 90) {
  console.log("優秀");
} else if (score >= 60) {
  console.log("及格");
}

原則： 條件判斷應該從最嚴格（範圍最小）的條件開始寫。

錯誤四：誤用假值判斷

容易出錯的範例：


let count = 0;  // 0 是假值

if (count) {
  console.log("有資料");
} else {
  console.log("沒有資料");  // 會執行這行，但 0 可能是有效的數量
}

正確寫法：


let count = 0;

if (count !== undefined && count !== null) {
  console.log("有資料");
} else {
  console.log("沒有資料");
}

五、重點整理

  • 基本語法：if 檢查條件，else 處理條件不成立的情況，else if 處理多個條件。
  • 條件表達式：必須使用 === 或其他比較運算子，不要誤用賦值運算子（=）。
  • 大括號習慣：即使只有一行程式碼，也建議使用大括號包圍，避免後續修改時出錯。
  • 條件順序：從最嚴格的條件開始寫，避免範圍較大的條件擋住範圍較小的條件。
  • 真值假值：空字串、0、null、undefined、NaN 都是假值，使用時需注意是否符合業務邏輯。
